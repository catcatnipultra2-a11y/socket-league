# socket-league
very real rocket league
for first thing you have to download and run the dependencies installer, its probably not needed if you already have python and ursina installed, but better to check, delete or remove the file "main.py" the dependencies installer generates, ignore that, it was just a test and i have no idea how to remove it nor have enough will power to.
INSTALL:put main.py (downloaded from this repo, not the one generated) in a folder, together with the audio files ball_light.mp3 ball_heavy.mp3 and goal.mp3, you can freely change the audio files to whatever you want, commands are: shift: boost
wasd: moving around
space: jump
c: change camera
1: bring ball infront of you
PS: dont try to double jump, it crashes the game, and i have no idea why
